> Vue.js 多页应用模板

## 开发起步

安装依赖的开发环境：nodejs yarn

``` bash
# 安装依赖
yarn install
```

### 启动开发服务器
``` bash
yarn run serve
```

### 构建项目到 dist目录
``` bash
yarn run build
```

### 测试
``` bash
yarn run test
```

### 运行代码检查
``` bash
yarn run lint
```

### 自定义配置
详见 [配置参考](https://cli.vuejs.org/zh/config/).


## 目录说明
* dist/: 构建后生成的目标文件夹
* src/: 源文件目录
* src/assets/: 图片字体等
* src/components/: 公用自定义组件
* src/pages/: 页面总目录
* src/pages/<Page Name>/: 每个页面的单页应用目录
